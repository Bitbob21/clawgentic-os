Häufige Fragen
Brauche ich technisches Wissen?
Nein. Sie brauchen nur einen Server (oder wir stellen einen bereit). Alles andere — Installation, DNS, SSL — macht Max für Sie.
Was passiert wenn ich kündige?
Sie behalten alle Ihre Daten als JSON-Export. Ihr Server bleibt bestehen — Founder OS lässt sich jederzeit wieder installieren.
Kann ich den Server selbst wechseln?
Ja. Founder OS läuft auf jedem Ubuntu/Debian-Server. Bei einem Wechsel hilft Max bei der Migration — als Add-on.
Wie funktioniert die Bezahlung?
Rechnung via Banktransfer oder Lightning Address. monatlich oder jährlich (10% Rabatt bei Jahreszahlung).
Was ist Max genau?
Max ist Ihr AI Chief of Staff — ein AI-Agent der auf Ihrem Server läuft und über Ihr Dashboard erreichbar ist. Kein Abo von OpenAI nötig.
Kann ich erst testen?
Ja. Managed und Concierge haben 14 Tage Testphase. Danach wird regulär abgerechnet. Self-Hosted können Sie direkt installieren.
